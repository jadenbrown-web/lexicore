# LexiCore

LexiCore is a browser-based German vocabulary study app built with plain HTML, CSS, and JavaScript.

## Project files

- `index.html` is the GitHub Pages entry file.
- `style.css` contains the app styling.
- `app.js` contains the quiz, word library, and theme logic.

## GitHub Pages

This repository is a static site. GitHub Pages should publish it from:

- Branch: `main`
- Folder: `/` (root)

When GitHub Pages is enabled, the site should be available at:

https://jadenbrown-web.github.io/lexicore/
